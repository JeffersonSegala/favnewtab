# FavNewTab
Extensão para o Chrome que substitui a nova aba padrão por uma visualização dos favoritos

# Política de Privacidade - Favorites New Tab

1. Coleta de Dados: Esta extensão acessa seus favoritos apenas para exibi-los na interface. Nenhum dado pessoal é coletado ou armazenado por nós.

2. Processamento de Dados: Todo o processamento de dados dos favoritos ocorre localmente no navegador do usuário. Não enviamos suas informações para servidores externos ou terceiros.

3. Alterações: Caso a extensão mude a forma como lida com dados no futuro, esta política será atualizada.
